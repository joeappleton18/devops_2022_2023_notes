# Assessment Overview

You are required to undertake a small-team (groups of two to four) web-based development project that requires you to make and carefully justify your choices surrounding key DevOps concerns. In doing so, you will develop an application based on an assessment scenario provided. Overall, this project forms the theme of the module’s two assessments: each assessment will count 50% towards your final grade

To get going with the assessments:

Read the assessment scenario

Read the assessment briefs

Watch the assessment overview video

Form a group, using the group selection tool on the SOL page (opens new window)
